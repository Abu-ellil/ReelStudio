## إضافة مقابض سحب (drag handles) لتحجيم القوائم الجانبية

الملف: `AI_Reel_Studio_Unified.html`

### الفكرة
التخطيط الحالي CSS Grid: `340px 1fr 320px` على `.layout` (سطر 24). سنسمح للمستخدم بسحب مقابض بين الأعمدة لتوسيع/تضييق القائمتين الجانبيتين (يسار ويمين)، مع حفظ العرض في `localStorage` ليبقى بعد إعادة التحميل. الوسط يبقى مرنًا (`1fr`).

### الخطوة 1 — CSS للمقابض (ضمن `<style>`، بعد سطر 24)
```css
/* مقابض السحب بين الأعمدة */
.gutter{position:relative;width:6px;cursor:col-resize;background:transparent;flex:0 0 6px;}
.gutter::before{content:'';position:absolute;top:0;bottom:0;left:50%;width:2px;transform:translateX(-50%);
  background:var(--border);border-radius:2px;transition:background .15s ease, box-shadow .15s ease;}
.gutter:hover::before,.gutter.dragging::before{background:var(--cyan);box-shadow:0 0 8px var(--cyan);}
body.gutter-dragging{cursor:col-resize;user-select:none;}
body.gutter-dragging *{pointer-events:none;}
```

تغيير `.layout` من grid إلى flex ليتسق مع المقابض (مقابض السحب أصبحت أبناء مباشرين):
```css
.layout{display:flex;gap:13px;align-items:flex-start;}
.layout > .col{flex:0 0 340px;min-width:0;}        /* يسار افتراضيًا */
.layout > .col.center{flex:1 1 0;min-width:0;}      /* وسط مرن */
.layout > .col.right{flex:0 0 320px;min-width:0;}   /* يمين افتراضيًا */
@media(max-width:1300px){.layout .gutter-r,.layout .col.right{display:none;}}
@media(max-width:1000px){.layout{flex-direction:column;}.layout .gutter{display:none;}
  .layout > .col{flex:0 0 auto;width:100%;}}
```
ملاحظة: نضيف classes (`col`/`col center`/`col right`) على الأعمدة الثلاثة (الأخطاط 222/430/457)، و`gutter`/`gutter-r` على المقابض.

### الخطوة 2 — تعديل الترميز (markup)
- العمود الأيسر (سطر 222): `<div>` → `<div class="col">`
- بعد العمود الأيسر (قبل CENTER): إدراج `<div class="gter gutter-l" data-side="left"></div>`
- العمود الأوسط (سطر 430): `<div>` → `<div class="col center">`
- بعد الوسط: إدراج `<div class="gutter gutter-r" data-side="right"></div>`
- العمود الأيمن (سطر 457): `<div>` → `<div class="col right">`

### الخطوة 3 — JavaScript لمنطق السحب + الحفظ
داخل `<script>` (قرب نهايته قبل `</script>`):
```js
// ======== resizable side panels ========
(function(){
  const layout=document.querySelector('.layout');
  if(!layout) return;
  const MIN=240, MAX=560, DEFAULTS={left:340,right:320};
  const leftCol=layout.querySelector('.col:not(.center):not(.right)');
  const rightCol=layout.querySelector('.col.right');
  const KEY='reelStudio.panelWidths';
  let widths={left:DEFAULTS.left,right:DEFAULTS.right};
  try{const s=JSON.parse(localStorage.getItem(KEY)||'{}');if(typeof s.left==='number')widths.left=clamp(s.left);if(typeof s.right==='number')widths.right=clamp(s.right);}catch(e){}
  apply();
  function clamp(v){return Math.max(MIN,Math.min(MAX,v));}
  function active(){return window.innerWidth>1300;}      // تعطيل تحت نقاط التوقّف
  function apply(){
    if(!active()){leftCol.style.flexBasis='';rightCol.style.flexBasis='';return;}
    leftCol.style.flexBasis=widths.left+'px';
    rightCol.style.flexBasis=widths.right+'px';
  }
  layout.querySelectorAll('.gutter').forEach(g=>{
    g.addEventListener('mousedown',e=>{
      if(!active())return;
      e.preventDefault();
      const side=g.dataset.side;
      g.classList.add('dragging');document.body.classList.add('gutter-dragging');
      const startX=e.clientX;
      const startW=side==='left'?widths.left:widths.right;
      const isRTL=(document.dir==='rtl'||document.documentElement.dir==='rtl');
      function move(ev){
        let dx=ev.clientX-startX;
        if(isRTL)dx=-dx;                                // عكس الاتجاه في RTL
        const w=clamp(startW+(side==='left'?dx:-dx));
        if(side==='left'){widths.left=w;}else{widths.right=w;}
        apply();
      }
      function up(){
        g.classList.remove('dragging');document.body.classList.remove('gutter-dragging');
        document.removeEventListener('mousemove',move);document.removeEventListener('mouseup',up);
        try{localStorage.setItem(KEY,JSON.stringify(widths));}catch(e){}
      }
      document.addEventListener('mousemove',move);document.addEventListener('mouseup',up);
    });
  });
  window.addEventListener('resize',apply);             // إعادة الضبط عند عبور نقاط التوقّف
})();
```

### الخطوة 4 — التحقّق
- فتح الملف في المتصفح، التأكد أن الماوس فوق الفاصل بين الأعمدة يُظهر مؤشر `col-resize` وشريطًا سماويًا.
- السحب لتوسيع/تضييق القائمتين؛ الوسط يعيد التدفّق تلقائيًا (canvas بـ max-width:100%).
- إعادة تحميل الصفحة → الأبعاد محفوظة.
- تصغير النافذة تحت 1300px → العمود الأيمن والمقبض الأيمن يختفيان؛ تحت 1000px → تكوّم عمودي.
- التأكد أن وضع RTL (عربي) يسحب بالاتجاه الصحيح.

### الحدود
- الحد الأدنى 240px، الأقصى 560px للقوائم الجانبية.
- الوسط لا يُحجَّم مباشرة بل يأخذ المساحة المتبقية.
- التعطيل التلقائي تحت نقاط التوقّف (1300px و1000px) لعدم التعارض مع media queries.
